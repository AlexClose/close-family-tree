🔧 Firebase Setup (One-Time, ~10 Minutes)

Follow these steps to connect your family tree to a free cloud database so everyone can see the same live data:

  1. Go to console.firebase.google.com and sign in with your Google account.
  2. Click "Add project" → name it close-family-tree → continue through the steps.
  3. In your project, click "Realtime Database" in the left menu → "Create Database" → choose "Start in test mode" → pick any location.
  4. Click the gear icon ⚙️ → "Project settings" → scroll down to "Your apps" → click the web icon </> → register the app → copy the firebaseConfig object shown.
  5. Open this HTML file in a text editor → find the section that says REPLACE_WITH_YOUR_API_KEY → paste in your config values.
  6. Save the file and upload to GitHub Pages — from that point on, all data saves to Firebase automatically and anyone with your link sees the same tree.

Once Firebase is connected, you can delete or hide this instructions section from the HTML file.
